# Screenshots

The main `README.md` references the images below. Capture them from the live
app (https://lumen-smoky.vercel.app) or your local `npm run dev`, save each
with the **exact filename**, drop them in this folder, then commit. The README
gallery will render automatically — no Markdown edits needed.

Recommended: full-window PNG, ~1600px wide, light traffic so personalized rows
are populated (sign in and rate the 10 onboarding films first).

| Filename | What to capture |
|---|---|
| `home.png` | The Home screen — hero billboard + Mood Dial + personalized rows |
| `mood-dial.png` | The 2D Mood Dial mid-drag with results streaming in |
| `why-card.png` | A title card flipped to show the 3 "Why this movie" reasons |
| `journal.png` | The Echo Journal — a logged film with its reflection question |
| `recap.png` | A Living Recap full-screen glass card |
| `title-detail.png` | A title detail page (cast/crew + "Why this for you") |
| `discover-timebox.png` | Time-Box Discovery ("I have N minutes" → ranked picks) |

`home.png` is also used as the hero image at the top of the README.
